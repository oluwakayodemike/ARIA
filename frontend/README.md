# ARIA Frontend

React, Vite, TanStack Router, React Query, and Tailwind CSS frontend for ARIA.

Most setup and demo instructions live in the root README:

```text
../README.md
```

Common commands:

```powershell
pnpm install
pnpm run dev
pnpm run lint
pnpm run build
```

The frontend expects the ARIA API at `http://localhost:8080` unless `VITE_API_BASE_URL` is configured.
